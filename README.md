# Atpay::Button::Generator

Atpay button generator library.

## Installation

Add this line to your application's Gemfile:

    gem 'atpay-button-generator'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install atpay-button-generator

## Usage

gem install atpay-button-generator
